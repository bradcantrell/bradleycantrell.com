# Sedimachine
**Year:** 2012
**Type:** Manifest / Prototype
**Location:** LSU
**Collaborators:** Frank Melendez
**Phase:** Phase II — The Operative Landscape (2012–2016)

## Description

A plexiglass surface inclined, water delivered via perforated copper tube, sand introduced into the flow to pattern based on which perforations were open. Phase 2 introduced twelve servo-actuated gates on a robotic spillway, each controlled by Arduino, enabling programmable sequences that choreographed water delivery. The sensing experiments with Kinect were largely unsuccessful: the depositional layer proved too thin for the Kinect's resolution to capture meaningful topographic variation.

## Contribution

Choreography over form: sediment patterning approached as a design problem not by sculpting deposits directly but by controlling flow conditions that produce them. The distinction between designing forms and designing the operations that generate forms. Productive failure: the failed Kinect experiments motivated search for modeling systems producing more substantial morphological change. Programmable hydrology: flow as a variable to be designed, not a constant to be observed.
