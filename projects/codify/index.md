# Codify
**Year:** 2018
**Type:** Publication
**Collaborators:** Adam Mekies (editor)
**Publisher:** Routledge
**Phase:** Phase III — Codification (2016–2020)

## Description

From using computation for landscape architecture to thinking computationally about landscape architecture. Generative design generating planting plans adapting to microclimates. Managing indeterminacy: code provides a genotype generating phenotypes that can adapt to changing conditions. Tool-making: landscape architects as tool-makers not tool-users, with literacy in Python, JavaScript, and Grasshopper. Richard Hindle: "Codify convincingly argues that Landscape Architecture is uniquely positioned to define this sector of technology, and in the process redefine itself."

## Contribution

A fundamental shift from computation for landscape to computational thinking about landscape. The landscape architect as algorithm curator: if ecological systems are too complex to fully specify, design operates through establishing rules and parameters within which outcomes emerge. Writing algorithms calibrated to specific ecological questions is a mode of inquiry, not merely a technical skill.
